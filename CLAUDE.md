## Build & Apply

```bash
# Full system rebuild (system + home-manager in one shot)
sudo nixos-rebuild switch --flake ~/nixos-config#nixos && sudo systemctl daemon-reload
alias: nixrebuild

# Home-manager only (standalone, for testing user changes without root)
home-manager switch --flake .#terrya
alias: homerun

# Update flake lockfile
nix flake update

# Garbage collection
sudo nix-collect-garbage -d
```

## Conventions

### Module Structure
- **System modules** (`system/`) receive `{ pkgs, ... }` or `{ config, pkgs, inputs, ... }`
- **User modules** (`user/`) receive `{ config, lib, pkgs, inputs, ... }`
- All modules get `inputs` via `specialArgs` in `flake.nix`

### Adding a new system-level feature
1. Create module in `system/<category>/<name>.nix`
2. Import it in `configuration.nix` under `imports`
3. If it needs a flake input, add to `flake.nix` and pass via `specialArgs`

### Adding a new user program
1. Create module in `user/<category>/<name>.nix`
2. Import in `home.nix` under `imports`
3. If it needs `lazyvimModule`, pass it as extra arg

### General Rules
- `hardware-configuration.nix` â€” never edit manually
- New categories go in `system/` or `user/` subdirectories
- Use `environment.systemPackages` for system-wide packages (in `configuration.nix`)
- Use `home.packages` for user-specific packages (in `home.nix`)
- Secrets use `/etc/secrets/` path (see crypttab entry for LUKS key)
- `allowUnfree = true` is set globally; extra unfree pkgs can be added to `allowUnfreePredicate`

### Mango WM Integration
- **System-level**: `programs.mango.enable = true` is in `configuration.nix:219`, provided by `mangowc.nixosModules.mango` from `flake.nix`
- **User-level**: `wayland.windowManager.mango.settings` in `user/WM/mangowm.nix` generates `~/.config/mango/config.conf`
- **Waybar config**: Maintained via stow in `~/.dotfiles/mangowc/.config/mango/config.jsonc` + `style.css`
- **DMS scripts**: Audio/notifications/night-light helpers in `~/.dotfiles/mangowc/.config/mango/scripts/`
- **Autostart**: `exec-once` managed by home-manager module; generates `~/.config/mango/autostart.sh`
- **Rebuild**: `nixrebuild` applies both system and user config atomically
- **Selecting Mango**: Available in display manager session list after rebuild
